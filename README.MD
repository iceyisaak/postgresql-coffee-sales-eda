# PostgreSQL-Coffee-Sales-EDA

###### Dataset: https://www.kaggle.com/datasets/najir0123/monday-coffee-sql-data-analysis-project/data
---

### Tech Stack
- **Database**: PostgreSQL
- **Language**: SQL (Advanced Joins, CTEs, Window Functions, Aggregations)
- **Concept**: Explorative Data Analysis (EDA)

---

### Objectives
In this project I analyzed coffee sales data using PostgreSQL to identify growth opportunities. I transformed raw transaction records into a strategic expansion plan, identifying the top three Indian cities for new physical locations based on proven consumer demand and historical sales performance.

---
### CONCLUSION & RECOMMENDATION

Based on the analysis, the top 3 cities that are worthy of business investment:

1. Pune: High avg sales/customers, Highest total revenue, Low rent/customer
2. Delhi: Highest number of coffee consumers, Highest total number of customers, Low rent/customer
3. Jaipur: Highest total number of customers, Low rent/customer, High sales/customer
